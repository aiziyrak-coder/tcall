## Android Studio Gradle xatolari

### "Project Migration Error" (Daemon JVM)
1. **OK** bosing — odatda build baribir ishlaydi
2. **File → Settings → Build, Execution, Deployment → Build Tools → Gradle**
3. **Gradle JDK** → **JDK 21** tanlang (yo'q bo'lsa **Download JDK 21**)
4. **File → Sync Project with Gradle Files**

### Gradle sync qizil xato

```bash
cd E:\Tcall.uz

```
